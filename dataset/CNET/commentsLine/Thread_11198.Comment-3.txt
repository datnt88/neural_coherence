all you should need is an ethernet card in each computer and cross-over ethernet cable for the connection.
both os's should be able to automatically configure the hardware so that sharing of files is possible.
the network setup wizard on xp should be run on both computers, but isn't absolutely necessary.
to run the xp setup wizard on 98 you can use the xp setup cd or a floppy disk with the wizard on it which can be created when the wizard is run on xp first.
